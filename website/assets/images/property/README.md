# Property photos

Original WhatsApp photos live in `_source/whatsapp/`. Older FINN screenshot crops are in `_screenshots/`.

Import new WhatsApp originals:

```bash
npm run import-photos
```

Regenerate screenshot crops (legacy):

```bash
npm run crop-photos
```
