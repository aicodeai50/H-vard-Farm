# Bilder for farm.legal

Legg ekte filer her, deretter oppdater `css/styles.css` (`.hero-bg` osv.).

## Anbefalte filnavn

| Fil | Bruk |
|-----|------|
| `hero-drone.jpg` | Forside hero (1920×1080+) |
| `gard-bygg.jpg` | Om gården, historie |
| `laave-selskap.jpg` | Selskap / låve |
| `bryllup-ute.jpg` | Bryllup |
| `bobil-omrade.jpg` | Bobilhotell |
| `logo.png` | Valgfri høyoppløselig logo |

## Etter opplasting

1. Erstatt Unsplash-URL i HTML/CSS med `assets/images/filnavn.jpg`
2. `git add website/assets/images/`
3. `git push` → auto-deploy

Maks filstørrelse anbefalt: under 500 KB per bilde (komprimer på squoosh.com).
