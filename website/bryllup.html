Skreddersøm fra første møte

Hvert bryllup er unikt. Vi hjelper dere med å finne flyt mellom seremoni, foto, middag og fest — og tilpasser lokaler og uteareal etter antall gjester og ønsker.

Med over 1 200 m² samlet under tak og store uteområder kan dere kombinere intim vielse med storslått fest — alt på ett sted.

  • Personlig planlegging
  • Fleksible lokaler
  • Bassengområde for sommerbryllup
  • Parkering og tilkomst

Ofte stilte spørsmål

  • Kan vi ha vielse utendørs? — Ja. Store grønne områder med innendørs alternativ ved dårlig vær.
  • Hvor mange gjester? — Kapasitet avhenger av opplegg — kontakt oss med antall, så foreslår vi løsning.
  • Overnatting? — Etter avtale i våningshus — spør i forespørselen.
  • Catering? — Dere velger leverandør eller vi kobler dere med lokale.
