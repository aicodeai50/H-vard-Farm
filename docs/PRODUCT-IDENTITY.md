# H-vard Farm — Product Identity

## What is H-vard Farm?

**H-vard Farm** is a **farm management platform** for Norwegian and Nordic agricultural operators. It helps farmers and farm managers plan fields, track crops, monitor weather context, document activities, and manage operational billing in one professional workspace.

## Problem we solve

Farmers juggle spreadsheets, paper notes, and disconnected tools for fields, harvest timing, and expenses. H-vard Farm centralizes **field records**, **crop planning**, **activity history**, and **account billing** so decisions are based on current data — not scattered files.

## Target audience

| Segment | Needs |
|---------|--------|
| **Farm owners** | Overview of all fields, harvest windows, costs |
| **Farm managers** | Daily operations, staff notes, weather-aware planning |
| **Agricultural companies** | Multi-field portfolio, team access, enterprise billing |

## Value proposition

> **One dashboard for your fields, crops, and farm operations — with transparent billing and room to grow.**

## Key features (MVP → v1)

1. **Farm overview** — Active fields, crop status, upcoming tasks
2. **Field & crop management** — Per-field crop type, area, planting/harvest dates
3. **Weather context** — Placeholder integration point for MET/Yr API
4. **Activity log** — Seeding, fertilizing, harvest, notes
5. **Billing & balance** — PayPal top-up, transaction history, ZENTRO sync
6. **Settings** — Profile, notifications, data export
