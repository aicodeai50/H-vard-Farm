## 1. Hovedmål

Utvikle en **moderne, eksklusiv og opplevelsesbasert** nettside som selger følelse — ikke bare bygninger.

**Produkter/tjenester som skal selges:**

| Område | Beskrivelse |
|--------|-------------|
| Bryllup | Drømmebryllup på landet, skreddersøm |
| Selskaper & arrangementer | Jubileer, konfirmasjon, dåp, firma, julebord, konserter |
| Gårdsopplevelser | Historie, natur, tradisjon |
| Bobilparkering | Søndre Haugen Bobilhotell (ute/inne/premium) |
| Fremtid | Gårdskafé, markeder, konserter, julemarked, gårdsbutikk |

**Merkevare-løfte:** *En historisk gård fra 1700-tallet med premium opplevelser.*

---

## 2. Stil og design

| Element | Retning |
|---------|---------|
| Følelse | Eksklusivt, moderne, historisk men luksuriøst |
| Bilder | Store dronebilder, gyllen time, gårdsmiljø |
| Farger | Mørke jordfarger, beige/krem, mørk grønn, diskrete gull-/røde aksenter fra logo |
| Typografi | Elegant serif til overskrifter; grønn skrift/cursive til «gård» der det passer |
| Logo | Stor, tydelig — sirkellogo med gårdillustrasjon, mønepryd, «EVENT • SELSKAP • BRYLLUP • FEST» |
| Teknisk | Mobil først, rask lasting, tydelige CTA-er |

**Unngå:** Generisk «gård-template», for mye tekst per side, stockfoto som ikke matcher Våler-gården.

## 4. Informasjonsarkitektur (sider)

### Forside
- **H1:** Historiske omgivelser for livets store øyeblikk  
- **Undertittel:** Bryllup, selskaper, arrangementer, gårdsopplevelser og bobilparkering i Våler.  
- **Hero:** Fullskjerm drone/video  
- **Knapper:** Send forespørsel · Book visning · Les mer  
- **Seksjoner under fold:** Korte teasere til Bryllup, Selskap, Bobil, Om gården, Nyheter

### Om gården
- Kort historie (1700-tallet), ro, natur, tradisjon  
- 2–3 emosjonelle avsnitt + 1 faktaboks  
- Bildegalleri / tidslinje (valgfritt)

### Bryllup (egen landingsside)
- Salg av drømmebryllup på landet  
- Vielse ute, middag inne, store områder, foto, skreddersøm  
- **CTA:** Send bryllupsforespørsel (egen skjema/type)

### Selskaper og arrangementer
- Målgrupper: jubileum, konfirmasjon, dåp, firmafest, julebord, konserter  
- Kapasitet / lokaler (låve 463 m²)  
- **CTA:** Send arrangementsforespørsel

### Bobilparkering (Søndre Haugen Bobilhotell)
- Rolig, landlig plassering; trygg lagring  
- **Tre nivåer** (fra eksisterende materiell):
  - **Haugen Ute** — fra 5 990 kr/år (fast plass, kamera, låst område)
  - **Haugen Inne** — fra 14 990 kr/år (oppvarmet hall, frostfritt)
  - **Haugen Premium** — fra 19 990 kr/år (+ batterivedlikehold, inspeksjon, prioritet)
- Pristabell etter kjøretøytype (tilhenger, caravan, bobil, veteranbil)  
- Tilleggstjenester: batteri, vask, vårklargjøring, dekkhotell, henting/levering  
- **Kontakt bobil:** Bobil@sondrehaugen.no · 901 98 671

## 5. Funksjonelle krav

- Responsivt design (mobil, nettbrett, desktop)  
- Kontaktskjema med routing (bryllup / arrangement / bobil / annet)  
- «Book visning» — kalenderintegrasjon eller enkel forespørsel  
- SEO: lokale søk (bryllup Våler, bobilhotell Østfold, gård arrangement)  
- GDPR-kompatibel personvern og informasjonskapsler  
- Google Maps / strukturert data (LocalBusiness, Event der relevant)  
- Språk: **norsk** primært; vurder kort engelsk under Bryllup for internasjonale gjester  

**Domene:** Pek **farm.legal** til produksjon; vurder redirect fra sondrehaugen.no om det skal være hovedmerke.

## 7. Suksesskriterier

1. Besøkende forstår på **5 sekunder** hva gården tilbyr  
2. Bryllup og selskap har **egne, selgende** sider med tydelig CTA  
3. Bobilhotell kan **bestille/forespørre** uten å ringe (skjema minimum)  
4. Siden oppleves **premium** — på nivå med inspirasjonssidene  
5. Ytelse: Lighthouse mobil ≥ 85 anbefalt
