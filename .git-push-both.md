# Push til begge GitHub-kontoer

| Remote | Repo |
|--------|------|
| **aicodeai50** | https://github.com/aicodeai50/H-vard-Farm |
| **havard50** | https://github.com/havard50/H-vard-Farm |

## En kommando (anbefalt)

```powershell
cd "c:\Users\sandr\OneDrive\Documents\Håvard-Farm"
.\PUSH-BOTH.ps1
```

`origin` er satt opp med **to push-URL-er** — `git push origin main` pusher til begge når begge kontoer er innlogget i Git Credential Manager.

## Manuelt

```powershell
git push "https://havard50@github.com/havard50/H-vard-Farm.git" main
git push "https://aicodeai50@github.com/aicodeai50/H-vard-Farm.git" main
```

## Token (aicodeai50 uten popup)

```powershell
$env:GITHUB_TOKEN_AICODEAI50 = "ghp_xxxx"
.\PUSH-BOTH.ps1
```

Token: https://github.com/settings/tokens → **repo** scope.
